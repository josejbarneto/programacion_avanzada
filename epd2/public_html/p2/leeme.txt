Para este ejercicio he creado una hoja de estilo para cada página en la carpeta HojasDeEstilo:

p3.html -> hojap3.css
galeria.html -> hojaGaleria.css
videos.html -> videos.css
formulario.html -> formulario.css

En la carpeta paginas se encuentran las páginas html.

Mi idea para el diseño de la página ha sido utilizar el atributo nav para crear un menú en común para todas las páginas,
El resto de los atributos los he modelado de manera distinta para cada una de las páginas.
